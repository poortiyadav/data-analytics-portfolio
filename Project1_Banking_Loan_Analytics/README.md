
Project 1: Banking Loan Analytics Dashboard

Tools: MySQL · SQL · Power BI (DAX) · Tableau · Advanced Excel
Domain: Banking & Financial Services — Loan Portfolio Management

Business Problem:
Branch managers and senior leadership needed a standardized view of loan portfolio health across branches, states, and product categories. Manual reporting was inconsistent. This project builds an end-to-end analytical solution.

KPIs Tracked:

Total Loan Amount Funded — measures total value of loans disbursed
Total Number of Loans — tracks loan volume
Total Collection — reflects repayment performance including principal and interest
Total Interest — captures revenue from loan interest
Branch-Wise Performance — analyzes revenue by branch (interest, fees, total)
State-Wise Loan Distribution — shows geographic concentration
Default Loan Rate (%) — proportion of defaulted loans to total portfolio
Delinquent Loan Rate (%) — percentage of loans overdue
Delinquent Client Count — tracks borrowers with missed payments
Grade-Wise Loan — assesses portfolio risk by borrower credit grade
Age Group-Wise Loan — categorizes loans by borrower age group
Loan Status Distribution — breaks down loans by active, delinquent, closed
Disbursement Trend — tracks changes in loan disbursements over time
No Verified Loans — identifies loans without proper verification
SQL Techniques Used:

Aggregate functions: SUM, COUNT, AVG
GROUP BY with ORDER BY for ranking
DATE_FORMAT for trend analysis
CASE statements for conditional logic
Percentage calculations for default and delinquency rates
